xlink:title

Title of a Link

May be used to title a link, for example by naming its target.

Remarks

Implementation of functionality related to this element is up to a receiving application. In web rendition, link titles will appear as tooltips in many browsers.

Attribute Values

In Element

<link>
Value Meaning
Text, numbers, or special characters A descriptive, human-readable title that acts as a brief description of the link
Restriction: This attribute may be specified if the element is used.
