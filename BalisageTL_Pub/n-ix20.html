Balisage: The Markup Conference

<listitem>

List Item

An individual list entry.

Expanded Content Model

(blockquote | equation | figure | informaltable | itemizedlist | mediaobject | note | orderedlist | para | programlisting | table | variablelist)+

Description

One or more of any of:

  • <blockquote> Block Quote
  • <equation> Equation
  • <figure> Figure
  • <informaltable> Informal Table
  • <itemizedlist> Itemized List
  • <mediaobject> Media Object
  • <note> Note
  • <orderedlist> Ordered List
  • <para> Paragraph
  • <programlisting> Program Listing
  • <table> Table
  • <variablelist> Variable List

This element may be contained in:

<itemizedlist>, <orderedlist>, <varlistentry>
