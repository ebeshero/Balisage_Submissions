<orderedlist>

Ordered List

A list in which each list item is numbered.

Attributes

continuation Numbering Continuation
numeration Numeration
startingnumber Starting Number

Expanded Content Model

(title?, listitem+)

Description

The following, in order:

  • <title> Title, zero or one
  • <listitem> List Item, one or more

This element may be contained in:

<appendix>, <article>, <blockquote>, <figure>, <listitem>, <note>, <para>, <section>, <td>, <th>
