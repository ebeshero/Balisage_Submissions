charoff

Table Character Offset

Percentage of column width to appear when align is set to “char”

Attribute Values

In Elements

<col>, <colgroup>, <tbody>, <td>, <tfoot>, <th>, <thead>, <tr>
Value Meaning Behavior
A number A numeric string that may include common punctuation (comma, colon, period, etc.) that specifies where to set the alignment
Restriction: This attribute is optional and is omitted if character alignment is not used.
