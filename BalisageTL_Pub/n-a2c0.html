<td>

Table Entry Cell

An entry in a table body.

Remarks

This tag set uses a modified HTML table model.

Attributes

align Alignment
char Table Cell Alignment Character
charoff Table Character Offset
colspan Column Span
rowspan Row Span
scope Scope
valign Vertical Alignment

Expanded Content Model

(#PCDATA | link | citation | emphasis | footnote | trademark | email | code | superscript | subscript | inlinemediaobject | itemizedlist | orderedlist | variablelist | equation | mediaobject | programlisting | note | para | quote | xref)*

Description

Any combination of:

  • Text, numbers, or special characters
  • <link> Link
  • <citation> Citation Reference
  • <emphasis> Emphasis
  • <footnote> Footnote
  • <trademark> Trademark
  • <email> Email
  • <code> Code
  • <superscript> Superscript
  • <subscript> Subscript
  • <inlinemediaobject> Inline Media Object
  • <itemizedlist> Itemized List
  • <orderedlist> Ordered List
  • <variablelist> Variable List
  • <equation> Equation
  • <mediaobject> Media Object
  • <programlisting> Program Listing
  • <note> Note
  • <para> Paragraph
  • <quote> Inline Quote
  • <xref> Cross-Reference

This element may be contained in:

<tr>
