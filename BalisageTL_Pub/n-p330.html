orient

Orientation

Specifies the orientation of a table, for example, landscape.

Remarks

This attribute is not available in the authoring DTD, as it will have no effect in web display. For backwards-compatibility and possible future use, it is retained in the production DTD.

Attribute Values

In Element

<table>
Value Meaning
land Lanscape orientation (90 degrees counter-clockwise from the rest of the text flow)
port Portrait orientation (the same orientation as the rest of the text flow)
Restriction: This attribute may be specified if the element is used.
