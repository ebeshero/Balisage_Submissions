<mathphrase>

Math Phrase

A mathematical expression that can be represented with ordinary text and a small amount of markup.

Expanded Content Model

(#PCDATA | inlinemediaobject | subscript | superscript | link | citation | emphasis)*

Description

Any combination of:

  • Text, numbers, or special characters
  • <inlinemediaobject> Inline Media Object
  • <subscript> Subscript
  • <superscript> Superscript
  • <link> Link
  • <citation> Citation Reference
  • <emphasis> Emphasis

This element may be contained in:

<equation>
