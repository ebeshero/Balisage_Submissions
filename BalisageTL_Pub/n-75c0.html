<term>

Term

A term or entry heading, to be described by its description element sibling.

Expanded Content Model

(#PCDATA | link | citation | emphasis | footnote | trademark | email | code | superscript | subscript | inlinemediaobject | quote | xref)*

Description

Any combination of:

  • Text, numbers, or special characters
  • <link> Link
  • <citation> Citation Reference
  • <emphasis> Emphasis
  • <footnote> Footnote
  • <trademark> Trademark
  • <email> Email
  • <code> Code
  • <superscript> Superscript
  • <subscript> Subscript
  • <inlinemediaobject> Inline Media Object
  • <quote> Inline Quote
  • <xref> Cross-Reference

This element may be contained in:
