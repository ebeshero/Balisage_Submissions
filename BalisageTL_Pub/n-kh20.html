<programlisting>

Program Listing

A verbatim environment for program source code or pseudo-code.

Attribute

xml:space Space

Expanded Content Model

(#PCDATA | emphasis | superscript | subscript)*

Description

Any combination of:

  • Text, numbers, or special characters
  • <emphasis> Emphasis
  • <superscript> Superscript
  • <subscript> Subscript

This element may be contained in:

<appendix>, <article>, <blockquote>, <figure>, <listitem>, <note>, <para>, <section>, <td>, <th>
