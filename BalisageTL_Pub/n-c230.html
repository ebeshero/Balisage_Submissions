linkend

Linkend

Points to an internal link target by identifying the value of its @xml:id.

Remarks

Available internal cross-referencing mechanisms, which use this attribute, are described more fully in the descriptions of the related elements, Cross-Reference, Citation Reference and Link.

Attribute Values

In Elements

<citation>
<link>
Value Meaning
An identifier (IDREF) The pointing uses the ID/IDREF mechanism, using the @linkend attribute of the associated element (for example, <xref>) to point to the @xml:id attribute of an existing element (for example, <section>).
Restriction: This attribute may be specified if the element is used.

In Element

<xref>
Value Meaning
An identifier (IDREF) The pointing uses the ID/IDREF mechanism, using the @linkend attribute of the <xref> element to point to the @xml:id attribute of an existing element.
Restriction: This attribute must be specified if the element is used.
