role

Role

On Emphasis, specifies the type of emphasis formatting.

On Keyword Set, indicates origin, authority or possible use.

Usage

When this attribute is not provided on Emphasis, a value of “ital” will be assumed.

On Keyword Set, this attribute is for editorial use and may be ignored.

Attribute Values

In Element

<emphasis>
Value Meaning
bital Bold italic font
bold Bold font
ital Italic font
rom Roman font
under Underscored font
Restriction: This attribute may be specified if the element is used.

In Element

<keywordset>
Value Meaning
Text, numbers, or special characters Identifies the function of the keyword set
Restriction: This attribute may be specified if the element is used.
