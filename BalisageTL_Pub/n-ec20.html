<informaltable>

Informal Table

A table to appear without caption or label.

Attributes

border Border (Frame) around a Table
cellpadding Cell Padding
cellspacing Cell Spacing
frame Table Frame
rules Rules

Expanded Content Model

((col* | colgroup*), thead?, tfoot?, tr+)

Description

The following, in order:

  • Any one of:
    • <col> Column, zero or more
    • <colgroup> Column Group, zero or more
  • <thead> Table Header, zero or one
  • <tfoot> Table Foot, zero or one
  • <tr> Table Row, one or more

This element may be contained in:

<appendix>, <article>, <blockquote>, <figure>, <listitem>, <note>, <para>, <section>
