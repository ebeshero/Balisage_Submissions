<colgroup>

Column Group

A container element to hold column descriptions (element <col>) for a table; may describe the default behavior for all columns with its attributes.

Remarks

This tag set uses a modified HTML table model.

Attributes

align Alignment
char Table Cell Alignment Character
charoff Table Character Offset
span Span
valign Vertical Alignment
width Width

Expanded Content Model

(col)*

Description

<col> Column, zero or more

This element may be contained in:

<informaltable>, <table>
