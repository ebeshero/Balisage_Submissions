<subscript>

Subscript

Subscripted text (in line).

Expanded Content Model

(#PCDATA | inlinemediaobject | superscript | subscript | link | citation)*

Description

Any combination of:

  • Text, numbers, or special characters
  • <inlinemediaobject> Inline Media Object
  • <superscript> Superscript
  • <subscript> Subscript
  • <link> Link
  • <citation> Citation Reference

This element may be contained in:

<citation>, <code>, <emphasis>, <link>, <mathphrase>, <para>, <programlisting>, <quote>, <subscript>, <subtitle>, <superscript>, <td>, <term>, <th>, <title>
