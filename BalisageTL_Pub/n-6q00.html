<equation>

Equation

A mathematical equation or expression, called out as a block and labeled.

Remarks

Simple math expressions may be provided as a Math Phrase, and more complex maths should be supplied as graphics, using Media Object. This tag set does not include MathML.

Attributes

xml:id XML Unique Identifier
xreflabel Cross-Reference Label

Expanded Content Model

((mathphrase | mediaobject)+, caption?)

Description

The following, in order:

  • One or more of any of:
    • <mathphrase> Math Phrase
    • <mediaobject> Media Object
  • <caption> Caption, zero or one

This element may be contained in:

<appendix>, <article>, <blockquote>, <figure>, <listitem>, <note>, <para>, <section>, <td>, <th>
