continuation

Numbering Continuation

Indicates how numbering should begin relative to an immediately preceding list.

Attribute Values

In Element

<orderedlist>
Value Meaning Behavior
continues Item/line numbering begins where the preceding list left off.
restarts Item/line numbering begins at “1” (formatted according to the list's @numeration).
Restriction: This attribute may be specified if the element is used.
