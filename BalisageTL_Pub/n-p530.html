pgwide

Page Wide

Identifies whether a figure can be the current paragraph width or require special handling

Remarks

This attribute has been included for possible future use, but it will not be respected when rendering Balisage papers as web pages.

Attribute Values

In Element

<table>
Value Meaning
0 The element should be rendered across the column.
1 The element should be rendered across the full text page.
Restriction: This attribute may be specified if the element is used.
