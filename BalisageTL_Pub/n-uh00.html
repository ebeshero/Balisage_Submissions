<col>

Column

The description of one column of a table.

Remarks

This tag set uses a modified HTML table model.

Attributes

align Alignment
char Table Cell Alignment Character
charoff Table Character Offset
span Span
valign Vertical Alignment
width Width

Expanded Content Model

EMPTY

Description

This is an EMPTY element

This element may be contained in:

<colgroup>, <informaltable>, <table>
