startingnumber

Starting Number

Indicates a number from which numbering an Ordered List should proceed.

Remarks

If given, this value will override any value given implicitly by @continuation.

Attribute Values

In Element

<orderedlist>
Value Meaning
A number A numeric string that typically does not include common punctuation (comma, colon, period, etc.)
Restriction: This attribute may be specified if the element is used.
